# 你的第一个 Metal 应用

在你的第一个 Metal 应用中，你要渲染的形状看起来更像是一个平面圆，而不是 3D 球体。这是因为你的第一个模型不包含任何透视或阴影。但是，其顶点网格包含完整的三维信息。
